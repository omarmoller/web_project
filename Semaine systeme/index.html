Who are we ?

Students at ENIT (National Engineering School Of Tunis), we are working on various projects in order to be part of the innovation in the field of IT using what we have learned this year. First of all, we started with installing all the software we need for the project developement. Then we moved to learning MPLABX basics which is written with c Language. Debugged and runned, we have simulated the code with virtual components step by step on PROTEUS PRO starting with a simple Led connection and ending with writing on an 20*2 LCD with a 4*4bites keyboard.

Projects

in the first project we launched a blinking led for a precise delay .the project's goal is getting familiar with coding with MAPLABX and simulating microcontrollers with PROTEUS.

in the second project we moved to a more sophisticated task which is connecting the PIC18F4550 microcontroller with a 2*20 LCD displayer and printing a simple message on its screen "Hello world!"

in the third project we added to the previous version a 4*4 bit keyboard and started printing pressed buttons from the keyboard on the LCD with a clearing option.
